MÈTODES D'ESDEVENIMENTS EN JQUERY

on() i off()

El mètode on() assigna un o més esdeveniments als elements seleccionats i elements secundaris

El mètode off(), normalment s'utilitza per eliminar els esdeveniments assignats amb el mètode on().

Per visualitzar el seu funcionament, clica en els botons on() i off()

one()

El mètode one() assigna un o més esdeveniments als elements seleccionats

Quan s'utilitza el mètode one(), lesdeveniment només s'executa UNA VEGADA per a cada element.

Per comprovar el seu funcionament, clica en el boto one() i apareix l'alert associat. Si tornes clicar en el boto, l'alert no apareix, ja que amb el mètode one(), l'esdeveniment associat només s'executa UN COP per a cada element.


click() i dblclick()

El mètode click() activa l'esdeveniment de clic o enllaça una funció per executar-se quan es produeix un esdeveniment de clic

El mètode dblclick() activa l'esdeveniment dblclick o enllaça una funció per executar-se quan es produeix un esdeveniment dblclick

mouseenter() i mouseleave()

L'esdeveniment mouseenter es produeix quan el punter del ratolí està sobre (entra) l'element seleccionat. El mètode mouseenter() activa l'esdeveniment mouseenter o enllaça una funció per executar-se quan es produeix un esdeveniment mouseenter

L'esdeveniment mouseleave es produeix quan el punter del ratolí abandona l'element seleccionat. El mètode mouseleave() activa l'esdeveniment mouseleave o enllaça una funció per executar-se quan es produeix un esdeveniment mouseleave

Per comprovar el seu funcionament, situa el punter sobre l'element i podras comprovar com aquest canvia. Quan retires el punter de l'element, aquest retorna a l'estat inicial

Mou el punter del ratolí sobre aquesta caixa

keydown() i keyup()

L'esdeveniment keydown es produeix quan es prem una tecla del teclat. El mètode keydown() activa l'esdeveniment keydown, o enllaça una funció per executar-se quan es produeix un esdeveniment keydown.

L'esdeveniment keyup es produeix quan es deixa anar una tecla del teclat. El mètode keyup() activa l'esdeveniment keyup o enllaça una funció per executar-se quan es produeix un esdeveniment keyup.

Per comprovar el seu funcionament, situa el cursor en l'input i pitja les tecles que vulguis. Els elements posteriors canvien quan prems i quan deixar anar la tecla

focus()

L'esdeveniment de focus es produeix quan un element s'enfoca (quan se selecciona amb un clic del ratolí o "navegant amb la pestanya" cap a ell). El mètode focus() activa l'esdeveniment focus o enllaça una funció per executar-se quan es produeix un esdeveniment focus

Per comprovar el seu funcionament, situa el cursor en l'input

Clica amb el punter del ratoli dins l'input si vols que tengui el focus: 

L'input ara té el focus

focusin() i focusout()

L'esdeveniment focusin es produeix quan un element (o qualsevol element que hi ha dins) s'enfoca. El mètode focusin() enllaça una funció per executar-se quan es produeix un esdeveniment de focus a l'element o qualsevol element que hi hagi dins.

L'esdeveniment focusout es produeix quan un element (o qualsevol element que hi hagi) perd el focus. El mètode focusout() enllaça una funció per executar-se quan es produeix un esdeveniment de focus a l'element o qualsevol element que hi hagi dins.

Per comprovar el seu funcionament, situa el cursor en l'input i després situa'l fora de l'input

resize()

L'esdeveniment de canvi de mida es produeix quan la finestra del navegador canvia de mida. El mètode resize() activa l'esdeveniment de canvi de mida o enllaça una funció per executar-se quan es produeix un esdeveniment de canvi de mida.

Per comprovar el seu funcionament, augmenta o disminueix el tamany de la finestra del teu navegador


scroll()

L'esdeveniment de desplaçament es produeix quan l'usuari es desplaça a l'element especificat. L'esdeveniment de desplaçament funciona per a tots els elements que es poden desplaçar i l'objecte finestra (finestra del navegador)

EFECTES EN JQUERY


fadeOut() i fadeIn()

El mètode fadeOut() canvia gradualment l'opacitat, per als elements seleccionats, de visible a ocult.

El mètode fadeIn() canvia gradualment l'opacitat, per als elements seleccionats, d'ocult a visible.

Per visualitzar el seu funcionament, clica primer en el boto fadeOut() i després en el boto fadeIn() i així successivament

FADE OUT <> FADE IN

fadeToggle()

El mètode fadeToggle() alterna entre els mètodes fadeIn() i fadeOut()

Per visualitzar el seu funcionament, clica successivament en el boto fadeToggle()

FADE TOGGLE

fadeTo()

El mètode fadeTo() canvia gradualment l'opacitat, per als elements seleccionats, a una opacitat especificada

Per visualitzar aquest canvi gradual d'opacitat, clica en el boto fadeTo(). En aquest cas s'ha configurat el canvi en 1000ms, de forma lineal per arribar a un 50% d'opacitat

FADE TO

slideUp() i slideDown()

El mètode slideUp() amaga els elements seleccionats. Aquest efecte es pot equiparar a l'acció d'obrir una persiana enrotllable

El mètode slideDown() mostra els elements seleccionats. Aquest efecte es pot equiparar a l'acció de tancar una persiana enrotllable

Per visualitzar el seu funcionament, clica primer en el boto slideUp() i després en el boto slideDown i així successivament

SLIDE UP <> SLIDE DOWN

slideToggle()

El mètode slideToggle() alterna entre els mètodes slideIn() i slideDown()

Per visualitzar el seu funcionament, clica successivament en el boto slideToggle()

SLIDE TOGGLE

animate()

El mètode animate() realitza una animació personalitzada d'un conjunt de propietats CSS.

Aquest mètode canvia un element d'un estat a un altre amb estils CSS. El valor de la propietat CSS es canvia gradualment per crear un efecte animat.

Podem configurar l'animació mitjançant valors relatius, valors absoluts o amb funcionalitat de cua.

Per visualitzar el seu funcionament, clica en els botons animate(valors relatius), animate(valors predefinits) i animate(funcionalitat de cua)

stop()

El mètode jQuery stop() s'utilitza per aturar una animació o efecte abans que acabi

Per visualitzar el seu funcionament, clica en el boto COMENÇA per iniciar l'animació, despres clica en el boto stop() per aturar-la (l'animació dura 10 segons)

 

hide() i show()

El mètode hide() amaga els elements seleccionats

El mètode show() mostra els elements ocults seleccionats

Per visualitzar el seu funcionament, clica en el boto hide() per amagar l'element i despres clica en el boto show() per mostrar-lo

 

callback

Les sentències de JavaScript s'executen línia per línia, de tal forma que durant l'execució d'un efecte, es possible passar a la següent linia de codi abans que l'efecte no hagi acabat, la qual cosa pot provocar errors.

Per evitar-ho, s'ha crear una funció callback. Aquesta funció s'executa un cop finalitzat l'efecte actual.

Per visualitzar el seu funcionament, clica en el boto ACTIVA OPACITAT i despres clica en el boto DESACTIVA OPACITA

encadenament

Amb jQuery es poden encadenar diversos mètodes. L'encadenament permet executar diversos mètodes jQuery (en el mateix element) dins d'una sola instrucció.

Per encadenar una acció, només cal afegir l'acció a l'acció anterior.

Per visualitzar el seu funcionament, clica en el boto INICIA ENCADENAMET (amb el boto RESET ENCADENAMENT tornam a la posició inicial)

Per visualitzar el seu funcionament, clica en els botons prepend() i append()

before() i after()

El mètode after() insereix contingut especificat després dels elements seleccionats

Per visualitzar el seu funcionament, clica en els botons before() i after()

Manipulació CSS
